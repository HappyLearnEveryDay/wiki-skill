# 花30分钟用AI做个小工具

**来源**：星辰技术专项组 | 2026年2月

## TL;DR

面向零编程基础用户的 AI 工具开发入门案例：用 [[../entities/trae]] 编辑器的 Solo 模式，30 分钟内用 Python 制作一个"查找并删除大文件"的桌面工具。强调 AI 使得"不会编程也能实现想法"。

## 核心主张

- 只需明确描述需求（功能、交付形式）即可让 AI 完成开发，无需编程基础。
- 使用 Trae Solo 模式：AI 自行分析、编写、调试，直到完成。
- 需求变更通过对话继续迭代；界面问题可截图粘贴发给 AI。
- 工具开发套件（Python 环境 + 编辑器）降低了环境配置门槛。

## Prompt 示例

```
功能：使用python开发一个文件清理工具
需求：1.选择文件夹，扫描所有文件
     2.按文件大小降序排序，显示前50个文件
     3.列表显示：文件名、路径、大小、复选框
     4.可打开选中文件和该文件所在目录
     5.可全选和取消全选，批量删除勾选文件，删除前二次确认
交付：1.打包成工具，可独立运行
     2.生成使用说明文档
```
