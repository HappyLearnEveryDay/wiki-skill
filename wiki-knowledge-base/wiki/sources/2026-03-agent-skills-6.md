# AI Agent 必装6个 Skills

**来源**：星辰技术专项组 | 2026年3月

## TL;DR

推荐了 6 个 [[../entities/claude-code]] 及 [[../entities/openclaw]] 等 AI Agent 的必装 [[../concepts/agent-skills]]，涵盖前端审美、文档处理、联网搜索、抗摆烂、长期记忆和元技能创建。核心洞察：通用技能在贬值，封装个人经验的能力在升值。

## 核心主张

### 6 个推荐 Skills

1. **Frontend Design**（Anthropic 官方，排名第一）
   - 硬性规范审美方向，禁止烂大街字体和紫色渐变。
   - 装与不装差距巨大，页面从千篇一律变为有设计感。

2. **办公四件套 docx/xlsx/pdf/pptx**（Anthropic 官方）
   - 提供 Word/PPT/Excel/PDF 完整读写代码模板。
   - 格式统一稳定，配合 Frontend Design 使用 PPT 颜值更佳。

3. **Web Access**（开源社区，@一泽，2000+ star）
   - 通过 Chrome DevTools Protocol 复用登录态访问站内内容。
   - 并行多 Agent 操作不同标签页；自动沉淀操作经验，越用越顺。

4. **PUA**（专治 AI 摆烂）
   - 四级压力升级，强制 AI 打断原地打转。
   - v3 内置多家公司方法论（阿里/字节/华为等），按任务类型自动选用。

5. **Claude-mem**（给 Agent 装长期记忆）
   - 三层渐进式检索（索引→时间线→完整细节），省 Token。
   - 支持 `<private>` 标签跳过敏感数据。

6. **Skill-Creator**（最重要的 Skill）
   - 帮用户自己构建个性化 Skill 的元工具。
   - **核心观点**：真正的必装 Skill 是封装你自己工作流和经验的那个——从消费者变成创造者。

### 核心洞察
- 通用能力正在被 Skill 替代，核心竞争力定义在变化。
- 知道自己需要什么、能把经验结构化成 Skill，是一种元能力。
- Skills 生态趋势：从单一工具到可组合、可叠加的复杂生态。
